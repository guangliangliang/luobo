# 守卫村庄 - 美术素材清单

## 项目信息

- 游戏名称：守卫村庄
- 游戏类型：2D 塔防游戏
- 引擎：Godot 4.x
- 推荐风格：卡通手绘、简约可爱、明亮配色、清晰黑色或深色描边
- 推荐视角：俯视或 45 度俯视
- 当前实际启用内容：野狼、强盗、大熊；箭塔、炮塔、冰冻塔；2 个关卡

## 统一生成要求

所有需要进游戏的素材都尽量满足：

```text
2D塔防游戏美术素材，卡通手绘风格，简约可爱，明亮配色，清晰轮廓，俯视或45度俯视视角，透明背景PNG，无棋盘格背景，无白底，无投影，主体居中，完整物体，适合Godot游戏精灵
```

注意：

- 必须是真透明 PNG，不要把棋盘格画进图片。
- 怪物先统一朝右或朝右下，因为当前游戏入口主要从左往右走。
- 不要生成超大画布。角色和塔建议每帧 `128x128` 或 `256x256`。
- 动画素材要保证每帧主体大小一致、脚底位置一致、朝向一致。
- UI 不建议生成整张界面图，优先生成按钮、面板、图标等可复用零件。
