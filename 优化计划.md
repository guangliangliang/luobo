# 守卫村庄 - 优化计划

## 📋 概述
基于用户需求，对"守卫村庄"微信小游戏进行页面和设计优化。

---

## ✅ 确认要做的优化

| 优先级 | 优化项 | 具体内容 |
|--------|--------|----------|
| 1 | 主菜单视觉增强 | 视差滚动效果 + 落叶+飞鸟动画 |
| 2 | 关卡选择页面 | 代码生成路线缩略图 + 星星显示 + 已通关标识 |
| 3 | 图鉴功能增强 | 点击塔/怪物图片可查看大图 |

---

## ❌ 确认不做的优化

| 优化项 | 原因 |
|--------|------|
| 删除启动画面 | 先不用改 |
| 设置面板扩展 | 不想加画质、控制、重置功能 |
| 结果页分享按钮 | 不需要分享 |
| 存档系统扩展 | 先不用保存关卡记录 |

---

## 📁 详细修改方案

---

### 1️⃣ `scripts/scenes/main_menu.gd` - 主菜单视觉增强

#### 新增功能

##### 1.1 视差滚动系统
- **功能**：
  - 监听鼠标/触摸移动事件
  - 根据输入位置，背景图轻微反向移动（营造深度感）
  - 移动端添加缓慢自动滚动效果

- **实现要点**：
  - 在 `_input()` 中监听 `InputEventMouseMotion` 和 `InputEventScreenDrag`
  - 记录当前鼠标/触摸位置
  - 根据位置计算背景偏移量（幅度控制在30-50像素）
  - 移动端：在 `_process()` 中用 `Time.get_ticks_msec()` 做缓慢的正弦曲线移动

##### 1.2 氛围动画系统
- **落叶效果**：
  - 用 `ColorRect` 或 `Sprite2D` 创建简单的叶子形状
  - 随机位置从顶部生成
  - 动画：下落 + 轻微旋转 + 透明度渐变
  - 叶子落地后自动销毁
  
- **飞鸟效果**：
  - 用简单的精灵或两个三角形模拟飞鸟
  - 随机位置从左侧生成
  - 动画：向右飞行 + 翅膀扇动效果
  - 飞出屏幕后自动销毁

#### 修改部分
| 函数 | 修改内容 |
|------|----------|
| `_ready()` | 添加视差和氛围系统初始化 |
| `_setup_ui()` | 修改背景设置，支持动态偏移 |
| `_process()` | 新增，处理动画更新 |
| `_input()` | 新增，处理鼠标/触摸输入 |

---

### 2️⃣ `scripts/scenes/level_select.gd` - 关卡选择页面优化

#### 新增功能

##### 2.1 代码生成关卡缩略图
- **实现方案**：
  1. 创建 `SubViewport`，尺寸约 `180x100`
  2. 在 SubViewport 中绘制：
     - 用 `bg_color` 填充背景（`ColorRect`）
     - 缩放 `path_points` 到缩略图尺寸，用 `Line2D` 绘制路线
     - 在村庄位置画一个小图标或圆点
  3. 将 `ViewportTexture` 显示在关卡卡片上

- **关键函数**：
  ```gdscript
  func _generate_level_thumbnail(level_data: LevelData) -> Texture2D:
      # 创建 SubViewport
      # 绘制内容
      # 返回纹理
  ```

##### 2.2 关卡卡片增强
- **显示内容**：
  - 顶部：关卡缩略图
  - 中间：关卡名称
  - 底部：星星 + 已通关标识

#### 修改部分
| 函数 | 修改内容 |
|------|----------|
| `_setup_ui()` | 修改关卡卡片创建逻辑，添加缩略图和星星 |
| 新增 `_generate_level_thumbnail()` | 生成关卡缩略图 |

---

### 3️⃣ `scripts/ui/codex_dialog.gd` - 图鉴查看大图功能

#### 新增功能

##### 3.1 大图查看弹窗
- **功能**：
  - 点击塔/怪物图片时触发
  - 创建全屏或大尺寸弹窗显示图片
  - 弹窗带关闭按钮（右上角或底部）

- **实现要点**：
  - 给每个头像按钮添加点击事件
  - 弹窗使用 `PanelContainer` + `TextureRect` + `Button`
  - 点击关闭按钮或背景遮罩关闭弹窗

#### 修改部分
| 函数 | 修改内容 |
|------|----------|
| `_make_avatar_button()` | 给图片添加点击事件连接 |
| 新增 `_show_large_image()` | 显示大图弹窗 |
| 新增 `_close_large_image()` | 关闭大图弹窗 |

## 🎨 设计规范

### 主菜单视觉
- 视差偏移幅度：30-50像素
- 落叶数量：同时显示3-5片
- 飞鸟出现频率：每2-4秒一只

### 关卡缩略图
- 尺寸：约 180x100 像素
- 路线宽度：3-5像素
- 村庄标记：一个小圆点或小图标
