# 《守护村庄》关卡背景和终点提示词

## 统一风格要求

所有素材建议保持同一套美术语言：

```text
2D塔防游戏美术素材，卡通手绘风格，简约可爱，明亮配色，清晰深色描边，俯视或45度俯视视角，适合Godot游戏使用
```

背景图建议：

- 尺寸：`1280x720` 或 `1920x1080`
- 不要角色
- 不要 UI
- 不要文字
- 画面中间区域保持干净，方便后续摆放道路、防御塔、怪物和终点
- 装饰元素尽量放在边缘和角落

终点建筑建议：

- 尺寸：`256x256` 或 `384x384`
- 透明背景 PNG
- 无棋盘格
- 无白底
- 无投影
- 主体居中
- 完整物体
- 45 度俯视视角
